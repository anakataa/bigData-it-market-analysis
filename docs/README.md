# Documentation

This folder contains project documentation, SQL queries, diagrams, and visual materials.

## What it does

It stores supporting files that explain how the project works and how the analysis was performed.

## Why this folder is needed

Documentation helps other people understand the project without reading every line of code. It also keeps Athena SQL queries, diagrams, and exported charts organized in one place.

## Important files and folders

- `athena_queries.sql` — SQL queries used for analysis in Amazon Athena.
- `Big Data/` — images, charts, and schema visuals used for project explanation.
