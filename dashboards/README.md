# Dashboards

This folder is used for dashboard screenshots and business intelligence visualizations.

## What it does

It stores visual results created from the processed job market data, for example salary charts, skill demand charts, or screenshots from Amazon QuickSight.

## Why this folder is needed

The project is not only about collecting data. It also needs to show the final insights in a simple visual form. This folder helps keep all dashboard-related files in one place, so the results are easier to present and review.
